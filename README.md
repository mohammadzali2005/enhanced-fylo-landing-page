## Overview
This project is an enhanced version of a Fylo landing page that I forked and improved. The goal was to refine the visual experience, improve layout consistency, and add subtle animations to make the UI feel more modern and engaging.

## Improvements made
- When I saw the project for the first time. I thought to myself that the header's height is a little bit to high and it makes the first view of user not very interactive, so I decreased the height of the header and also made the image in hero section a little bit smaller so the texts in hero section are more visible to users at the first glance.
- I also **changed the animation in header's items** and I tried to make it more modern.
